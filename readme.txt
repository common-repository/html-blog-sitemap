Adds a HTML sitemap of your blog posts by entering the shortcode [pwh_blog_sitemap].

== Description ==

Simply create a page (or post) and include the shortcode [pwh_blog_sitemap] onto that page. As user click through categories, the lists displays the clicked on category as the new "main" category.

SUPPORT: please use the product link below and post a comment on that page if you need support.

For the latest information visit [http://www.php-web-host.com/html-blog-sitemap-wordpress-plugin/](http://www.php-web-host.com/html-blog-sitemap-wordpress-plugin/)

We needed, and could not find a plugin to list blogs posts in logical categories for easy navigation, eg:

Blogs
-----
	Products
	--------
    		- Widget A
          	- Widget B

          	Product Sub Category
          	--------------------
           		- Blue Widget
           		- Red Widget
 
        Services
	--------
		- Service One
		- Service Two

== Installation ==
1. Copy the entire directory from the downloaded zip file into the /wp-content/plugins/ folder.
2. Activate the "HTML Blog Sitemap" plugin in the Plugin Management page.
3. Add the shortcode [pwh_blog_sitemap] to the page(s) of your choice.
